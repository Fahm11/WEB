Entities
Entities atau yang biasa dikenal HTML entity adalah rangkaian karakter yang ditempatkan dalam teks dokumen
HTML untuk mencetak karakter khusus yang mungkin ditafsirkan sebagai kode HTML oleh browser, atau karakter
khusus lainnya yang sulit dibuat dengan papan ketik biasa.

Breakline HTML
adalah tag yang digunakan untuk menyisipkan baris baru atau jeda baris dalam elemen HTML. 
yang di simbolkan dengan tag <br>

Horizontal Line 
Garis horizontal dalam HTML adalah garis yang dibuat menggunakan tag <hr>.
Tag ini berfungsi untuk memisahkan konten secara visual, seperti paragraf, pada halaman web
