HTML
HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat halaman website
dan aplikasi web. HTML merupakan bahasa pengkodean yang berisi kombinasi teks dan simbol yang dibuat 
berdasarkan format standar kode internasional atau ASCII.

Tag
Tag HTML adalah kata atau huruf yang diapit oleh tanda kurung siku (< dan >) yang berfungsi 
untuk mengatur dan mendefinisikan struktur, konten, dan tampilan halaman web. 
Tag HTML merupakan elemen penting dalam bahasa markup HyperText Markup Language (HTML)

Tag H
Tag <h> dalam HTML atau tag heading adalah elemen HTML yang digunakan untuk membedakan judul dan 
subjudul halaman web dari konten lainnya.

Tag P
Tag <p> dalam HTML adalah tag yang digunakan untuk mendefinisikan paragraf.
